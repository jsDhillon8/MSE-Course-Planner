# SFU MSE Course Navigator (Phase 1 Skeleton)

Starter web app for the updated Course Navigator:

- Multiple curriculum pages (pre-2024, post-2024, double degree)
- 4-year and 5-year templates
- 5-year option switcher (A/B/C)
- Planner grid + course details panel
- Course equivalency section in details view

## Run locally

```bash
npm install
npm run dev
```
